#### 오버로딩 vs 오버라이딩

- 오버로딩 (overloading) : 기존에 없는 새로운 메서드를 정의하는 것 (new)
- 오버라이딩 (overriding) : 상속받은 메서드의 내용을 변경한느 것 (change, modify)



#### Super

- 자손 클래스에서 조상 클래스로부터 상속받은 멤버를 참조하는데 사용되는 참조 변수.

``` java
class SuperTest{
    public static void main(String arg[]){
        Child c = new Child();
        c.method();
    }
}

class Parent{
    int x = 10;
}

class Child extends Parent{
    int x =20;
    void method(){
        sysout(x);			// x = 20
        sysout(this.x);		// x = 20
        sysout(super.x);	// x = 10
    }
}
```
