## Python Dictionary (Hash)

- Dictionary 란?

  - Key , Value로 이루어진 배열.
  - 중복되는 Key값이 들어오면 마지막으로 저장되는 Key값으로 대체 됨. 따라서 Key값은 고유해야 함.

- 쓰는 이유 : 

  - 변수들을 더 잘 활용하기 위하여
  - 하나의 변수에 더 많은 정보를 담기 위해서 씀.

- 활용 : 

  - value 값 안에다가 뭐든지 넣을 수 있음 
    - {key : value}라던지 
    - 배열이라던지
  - for a in b 
    - b에서 순회해서 데이터를 a에 저장.
    - for문의 scope는 indent로 구분
