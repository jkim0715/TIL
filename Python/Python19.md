### 유효성 검사

- HTML (1단계)

  - required, min_length 등

  - 브라우저가 막아줌
  - 근데 이번 수능 점수마냥 쉽게 뚫림

- Django Form (2단계)

  - is_valid()
    - 거의 여기서 걸러짐
  - blank = True 로 패스가능
  - CharField / TextField 는 blank =True 로 DB까지 패스가능
    - 빈값으로 놔두면 DB에 null이 들어가는게 아니고  빈 문자열이 들어가게 됨

- DB (3단계)

  - null = True 로 패스 가능.
  - Django 기본 값으로 null = false 로 되어있음.
