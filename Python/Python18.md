### venv 폴더

- 터미널을 가상환경을 만듬
  
  ```shell
  $python -m venv venv
  ```
  
  
  
- 터미널이 가상환경을 보도록 만듬 
  
  ```shell
  $source venv/Scripts/activate
  ```
  
- F1 -> Select Interpreter -> .`\venv\ 선택

- `$pip freeze`
  
  - install된 내역 버전까지 상세보기 



### requirements.txt(국룰)
