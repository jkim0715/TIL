Stack

- ```java
  Stack s = new Stack();
  
  //insert
  s.push();
  s.push();
  
  //check empty
  while(!s.isEmpty()){
      //pull out    
      System.out.println(s.pop());
  }

Queue (FIFO)

- ```java
  Queue q = new LinkedList();
  
  // Insert
  q.offer("1");
  q.offer("1");
  q.offer("1");
  
  //check Empty
  while(!q.isEmpty()){
      //Pull out 
      System.out.println(q.poll());
  }
  ```
